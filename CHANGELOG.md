# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-23

### Added
- Initial project structure for LexHarmoni.
- Corpus management for active and historical Indonesian financial regulations.
- FastAPI backend skeleton with health and manifest endpoints.
- Next.js frontend skeleton with landing page.
- Comprehensive OSS documentation (LICENSE, NOTICE, CONTRIBUTING, SECURITY, CODE_OF_CONDUCT).
- Automated license header script for Python files.
- Manifest validation script for regulatory metadata.
- Ground truth dataset for regulatory inconsistency benchmarks.
