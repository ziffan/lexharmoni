# Security Policy

## Supported Versions

The following versions of LexHarmoni are currently supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

**DO NOT open a public GitHub issue for security vulnerabilities.**

If you discover a security vulnerability, please report it via email to:
**kampusmerah.com@gmail.com**

We expect to acknowledge your report within **48 hours**.

Please include:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact.
- Suggested fix (if any).

We follow **coordinated disclosure**: We ask for 90 days from the initial report before public disclosure, to allow for investigation and fixing.
